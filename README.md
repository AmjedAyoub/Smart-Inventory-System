# Smart-Inventory-System
## Overview
The main purpose of this program is to manage restaurantsâ€™ inventory. The program uses Artificial Intelligence by taking historical data related to sales, accounting, distribution, and inventory to create forecasting planning related to existing data. Using an efficient programming language C#, the program predicts daily, and yearly patterns and activities to suggest ways to manage the availability of goods, reduce food costs, and gain greater visibility of inventory.
